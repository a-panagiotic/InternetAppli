# InternetAppli

CTGOV-03: Χώρες στις οποίες έχουν λάβει χώρα Κλινικές Μελέτες γύρω από μία συγκεκριμένη Ασθένεια

Να φτιάξετε μια δικτυακή εφαρμογή η οποία παρουσιάζει όλες τις χώρες στις οποίες έχουν λάβει χώρα κλινικές μελέτες για μία συγκεκριμένη Ασθένεια. Το input θα είναι το όνομα της ασθένειας το οποίο θα το ορίζει ο χρήστης μέσω της web σελίδας και το output θα είναι οι χώρες ταξινομημένες ανάλογα με τον αριθμό των κλινικών μελετών που έχουν λάβει χώρα ο οποίος θα πρέπει επίσης να παρουσιάζεται.

Σημείωση : Μέχρι στιγμής μπορώ να διευκρινήσω πως δεν θα χρησιμοποιηθεί κάποιο εργαλείο που δεν διδάχθηκε.
